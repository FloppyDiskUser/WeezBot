## Weezbot
weezbot is s Pokemon Go bot. It farms pokestops, transfers unwanted pokemon for candies, etc.

### Functions
```sh
- Graphical Interface
- Farming Pokestops
- Catching Pokemon
- Sniping Pokemon
- Select Position by Address
- Integrated Pokemon Nests ( by pkmngotrading.com )
- Easy To Use Interface
- Supports Multibot
```
